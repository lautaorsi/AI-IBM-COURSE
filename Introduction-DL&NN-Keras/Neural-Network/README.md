# Redes Neuronales (Artificales)

La idea detrás de las Redes Neuronales Artificiales está en agrupar en dsitintas capas un conjunto de Neuronas Artificiales (también llamadas Perceptrones), estas capas van a tener distintas funciones según el modelo, pero en general tenemos el siguiente esquema:

-   Input Layer

-   Hidden Layer(s)

-   Output Layer
