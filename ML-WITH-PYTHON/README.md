# Anotaciones Generales

## [Aprendizaje Supervisado](./Supervised-Learning/)

Se basa en la utilización de datasets etiquetados para entrenar algoritmos a clasificar o predecir nuevos datapoints. Es gracias a estos inputs/outputs etiquetados que el modelo puede acertar su precisión y "aprender".

## [Aprendizaje No Supervisado](./Unsupervised-Learning/)

Se basa en el análisis y clusterizacion de datasets no categorizados. La idea está en descubrir patrones en la información sin intervención humana.

## [Reducción de Dimensionalidad](./Dimensionality-Reduction/)

Es la idea de reducir la cantidad de features itnentando preservar la mayor cantidad de información posible. Muchas veces esto permite visualizar con mayor facilidad los patrones subyacentes de los datasets.

## [Evaluación y Validación de Modelos](./Evaluating-Validating-Models/)

Una vez armado el modelo se necesita poder saber que tan eficaz y preciso es, para esto se empelan distintas métricas según el tipo de modelo para obtener resultados consistentes.

## [Generalizibilidad de Modelos](./Model-Generalizability/)

En escenarios reales es importante tener la posibilidad de trabajar de forma fluida y sistemática sobre distintas partes del proceso, para esto se utilizan pipelines y optimizaciones de hiperparametros.
