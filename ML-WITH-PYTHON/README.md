# Anotaciones Generales

## [Aprendizaje Supervisado](./Supervised-Learning/)

Se basa en la utilización de datasets etiquetados para entrenar algoritmos a clasificar o predecir nuevos datapoints. Es gracias a estos inputs/outputs etiquetados que el modelo puede acertar su precisión y "aprender".

## [Aprendizaje No Supervisado](./Unsupervised-Learning/)

Se basa en el análisis y clusterizacion de datasets no categorizados. La idea está en descubrir patrones en la información sin intervención humana.
