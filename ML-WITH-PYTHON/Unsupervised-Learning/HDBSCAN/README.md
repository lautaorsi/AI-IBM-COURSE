# HDBSCAN

No requiere interacion humana y se basa únicamente en la estabilidad de los clusters (La perstitencia de un cluster sobre un rango de distancia)

## Parametros
No tiene

## Pseudo-algoritmo
1.  Combina clustering aglomerativo y de densidad
2.  Identifica cada punto como su propio cluster
3.  Aglomera los clusters jerarquicamente
4.  Se simplifica a un arbol con clusters estables
