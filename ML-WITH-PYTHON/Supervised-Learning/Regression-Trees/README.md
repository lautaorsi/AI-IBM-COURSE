# Sobre Arboles de Regresion

## ¿Para que sirven?
Sirven para predecir valores numericos continuos de variables (a diferencia de los arboles de decision que simplemente deciden).

## Ventajas
La mayor ventaja que tiene por sobre otras regresiones es su facil lectura y comprensión, la estructura arbolada y splits con criterio fijo permiten reconcer facilmente que camino lleva a la prediccion y en base a que decisiones.


### Criterion:
La forma de medir la calidad de un split
