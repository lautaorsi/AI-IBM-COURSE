# Supervised Learning

Tenemos dos tipos de objetivos en el aprendizaje supervisado

## Clasificación
Usan un algoritmo para asignarla a datapoints categorias específicas, dentro de los posibles algoritmos de clasificación tenemos:

-   [KNN](./KNN/)
-   [SVM](./SVMs/)
-   [Arboles de Decisión](./Decision-Trees/)
-   [Random forests](./Random-Forest/)

## Regresión
Usa un algoritmo para establecer relaciones de dependencia entre una variable objetivo y features, a fin de predecir valores numericos (que luego pueden ser interpretados de distintas maneras), dentor de los posibles algoritmos de regresión tenemos:

-   [Regresión Lineal Simple](./Linear-Regression/)
-   [Regresión Lineal Múltiple](./Linear-Regression/)
-   [Regresión Logística](./Logistical-Regression/)
-   [KNN](./KNN/)
